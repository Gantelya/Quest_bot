# 🎮 Quest Rooms Bot — Telegram-бот администратора квест-комнат
![Python](https://img.shields.io/badge/Python-3.9+-blue?style=for-the-badge&logo=python)
![python-telegram-bot](https://img.shields.io/badge/python--telegram--bot-20.x-green?style=for-the-badge)
![SQLite](https://img.shields.io/badge/SQLite-3-blue?style=for-the-badge&logo=sqlite)

> Telegram bot that fully replaces a quest room administrator. 
> Built with Python, python-telegram-bot and SQLite.

Бот берёт на себя 100% рутины администратора: принимает брони 24/7, 
отправляет напоминания клиентам, ведёт расписание и уведомляет владельца — 
без единого звонка и без найма сотрудника.


---

## ✨ Функции

| Функция | Описание |
|---|---|
| 📅 **Бронирование** | Пошаговый выбор квеста, даты, времени, ввод контактов |
| 📋 **Мои брони** | Просмотр и отмена бронирований прямо в боте |
| 🔔 **Напоминания** | Автоматически за 24 часа и за 2 часа до игры |
| ❓ **FAQ** | 8 готовых ответов на популярные вопросы |
| 📞 **Контакты** | Быстрый доступ к реквизитам компании |
| 🛡 **Уведомления** | Администратор получает уведомление о каждой брони/отмене |
| 📊 **Расписание** | Команды `/today` и `/week` для просмотра брони |

## 🚀 Установка и запуск

### Шаг 1 — Создайте бота

1. Откройте Telegram и напишите [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot`
3. Придумайте имя и username бота
4. Скопируйте полученный **токен**

### Шаг 2 — Узнайте свой Telegram ID

Напишите боту [@userinfobot](https://t.me/userinfobot) — он пришлёт ваш ID.

### Шаг 3 — Настройте `config.py`

Откройте файл `config.py` и заполните:

```python
BOT_TOKEN = "ВСТАВЬТЕ_ТОКЕН_СЮДА"          # от @BotFather
ADMIN_CHAT_ID = 123456789                   # ваш Telegram ID

CONTACT_INFO = "📞 +7 (999) ..."            # ваши контакты
ADDRESS = "ул. Ваша, д. 1"                  # ваш адрес
```

Настройте квест-комнаты в словаре `QUESTS`:
- `name` — название комнаты
- `description` — описание для клиента
- `price` — цена в рублях
- `time_slots` — доступное время (список строк `"ЧЧ:ММ"`)

### Шаг 4 — Установите зависимости

```bash
pip install -r requirements.txt
```

### Шаг 5 — Запустите бота

```bash
python bot.py
```

## ⚙️ Команды администратора

| Команда | Описание |
|---|---|
| `/today` | Все бронирования на сегодня |
| `/week` | Бронирования на 7 дней вперёд |

> Команды работают только для вашего аккаунта (по `ADMIN_CHAT_ID`).

---

## 🔧 Настройка расписания

Чтобы добавить или убрать временные слоты — отредактируйте `time_slots` в `config.py`:

```python
"time_slots": ["10:00", "12:00", "14:00", "16:00", "18:00", "20:00"],
```

## 📦 Деплой на сервер (опционально)

Для круглосуточной работы запустите на VPS через `systemd` или `screen`:
